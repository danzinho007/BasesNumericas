Sistema Binário : 0 1

Decimal = Binário 
0 = 0   - 5 = 101  - 10 = 1010 
1 = 1   - 6 = 110  - 11 = 1011
2 = 10  - 7 = 111  -
3 = 11  - 8 = 1000 -
4 = 100 - 9 = 1001 -

Binário > Decimal 
10      = 1 x 2¹  + 0 x 2°
        = 2 + 1
        = 3

1011    = 1 x 2³ + 0 x 2² + 1 x 2¹ + 1 x 2°
        = 1 x 8  + 0 x 4  + 1 x 2  + 1 x 1
        =   8    +   0    +   2    +    1
        = 11
